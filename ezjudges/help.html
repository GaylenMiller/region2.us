EZ Judges

How do I get started?

EZ Judges has four steps for startup

These steps are presented together on the “Easy Steps” tab

  1. 1Clear All
  2. 2Import Tournament
  3. 3Check Patterns
  4. 4Check Rounds

The “Clear All” deletes the judges and debate teams for any prior tournaments

The “Import Tournament” imports the registered judges, the parents, the round times and the room counts. The import file comes from the NCFCA Leader Dashboard for the tournament. Go to the “Reports” screen and download the “Judge Tracking 2014” file.

Note: The import step does not automatically “Clear” because this allows the judges to get imported again during the tournament to get any judges registered after the last import.

The “Check Patterns” step is to complete with five checks.

  1. 1Tournament Name
  2. 2Debate Elim Rounds and Rooms
  3. 3Debate Judge Count
  4. 4Speech Patterns A and B Events
  5. 5Speech Judge Count

Normally all the tournament information comes in correctly and the only work is on check #4 to drag and drog the speech events to the correct pattern.

The “Check Rounds” is to verify the round dates and times. If a round included or excluded incorrectly, drag and drop it to the correct location.

What is the purpose of the “Easy Tasks” tab?

It is presenting the options needed for a point in time

The tournament is broken into “Points In Time” based on the events and the rounds. Every debate round has a before and after point in time and each speech round has a before and after point in time. There is also a “Before Tournament” and “After Tournament” point in time.

Based on the current time, the appropriate “Point In Time” is shown. Move forward or backwards in time with either the slider or the left/right arrows. Click the circular green arrow to return the present.

What are some common problems?

A new judge was entered but EZ Judges is showing is a blank line in the room

This is a bug that refuses to show itself to the programmer and still exists. Normally new judges show up fine when testing the software, but in production they can sometimes become invisible. The work around is to exit EZ Judges and start it again.

If this bug in encountered, please send in feedback via the feedback option on the bottom right side of the screen

The database opening in read only mode

This can happen when EZ Judges is already open on your computer or, if the database file is shared, open on another computer.

The debate team name is incorrect

Update the team name in C-Tap Debate. The name in EZ Judges will be updated with the next debate file import.

A judge team name is incorrect

The ”Judges“ tab allows judge name updates. The judges are listed in alphabetical order. Click in the cell, to update the name and then click on another row to save the change.

Debate round was imported twice with different room assignments

The debate import adds to the existing information. If a debate round was imported, then the rooms changed in C-Tap debate and then imported again, the two sets will be merged in EZ Judges. This can be corrected by dropping the round in EZ Judges and importing the debate round again. The ”Drop Debate Round“ can be done either on the ”Easy Tasks“ tab, which allows the dropping of either the LD or the TP round, or on the ”Rooms“ tab, which will drop both events from the debate round selected in the drop down.

Are there any additional features?

Judges Activity Report

The “Judges” tab includes the “Judges Activity Report” which shows the events and rounds judged by the currently selected judge in the list of judges.

Top Judges Report

The “Judges” tab includes the “Top Judges Report” which lists the judges by the number of events judged.

Judges Thank You Export

The “Judges” tab includes the “Judge Thank You Export” which will exports to Excel all the Judges that judged an event. The goal is to thank the judges that showed up and judged and leave out the ones that signed up but did not judge.

Tournament Graphic Update

The “Reference – Tournamet Graphic” tab allows a new tournament graphic to be loaded from a file.Use a graphic that is twice as long as it is high and is a Windows bitmap format (.bmp).

Judge Count Additional Options

The “Reference – Choice Lists” tab allows a new Debate or Speech Judge Count to be entered. Click on the desired judge count entry in the top half of the screen to show the existing entries in the bottom half of the screen. Click on the last entry and error down to get a blank row. Fill in the code, descripion and the order it should appear in the drop down list. After entry, click on the previous row to save the change and then restart EZ Judges to see the new entry in the drop down list on the Patterns tab.

For example, a new code might be ”1337“, meaning 1 judge for prelims, 3 for quarter finals, 3 for semi-finals and 7 for finals.
